Servicios Api Rest Jaguar

Noticias

  • getNoticias
    endpoint: GET: /api/nots
  • getNoticia
    endpoint: GET: /api/nots/id
  • nuevaNoticia
    endpoint: POST: /api/nots {body}
  • actualizarNoticia
    endpoint: POST: /api/nots/actualizarNoticia/id {body}
  • borarNoticia
    endpoint: DELETE: /api/nots/id

Usuarios

  • login
    endpoint: POST: /api/nots/login {body}
  • login
    endpoint: POST: /api/users/login {body}

Subir Imagenes

  • Imagen Noticia
    endpoint: POST: /api/uploads/noticia/id {req.files-noticiaImagen}
  • Imagen Banner
    endpoint: POST: /api/uploads/banner {req.files-bannerImg}
  • Borrar Banner
    endpoint: POST: /api/uploads/banner/id {req.text-nombreBanner}
  • Get Banners
    endpoint: GET: /api/uploads/get-banners
  • Imagen Beneficio
    endpoint: POST: /api/uploads/beneficio/id {req.files-beneficioImg}

  • Auth POST

  • UserInfo POST

  • UserInfo POST
    endpoint: POST: /api/soffid/revoke
